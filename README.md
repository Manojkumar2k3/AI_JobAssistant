# 🚀 AI Job Assistant

An AI-powered tool that compares resumes with job descriptions and provides a match score along with skill insights.

---

## 🔍 Features

- 📄 Upload resume (PDF)
- 📊 Match score (0–100%)
- ❌ Missing skills
- ✅ Strengths
- 🤖 AI-based semantic matching (LLM)

---

## ⚙️ Tech Stack

- FastAPI (Backend)
- Streamlit (Frontend)
- Groq LLM (AI)
- pdfplumber (Resume parsing)

---

## 🧠 How It Works

1. Upload resume + paste job description  
2. Resume is converted to text  
3. AI compares resume with job description  
4. Returns match score + insights
